# E-Commerce Sales Analysis Dashboard

This project analyzes e-commerce sales data to identify key business trends, customer purchasing behavior, product performance, and revenue patterns. The dataset was cleaned and processed using Python and Pandas, followed by exploratory data analysis to uncover insights related to sales growth, profit, orders, customer segments, and product categories.

An interactive Power BI dashboard was developed to visualize important KPIs such as total sales, total profit, total orders, monthly revenue trends, top-performing products, and customer behavior patterns. The dashboard helps businesses monitor sales performance, identify growth opportunities, and make data-driven decisions.

## Key Features

- Cleaned and transformed raw e-commerce sales data using Python and Pandas
- Performed exploratory data analysis to identify revenue trends and customer behavior
- Analyzed product performance, monthly sales trends, and purchasing patterns
- Built an interactive Power BI dashboard with KPIs and filters
- Visualized total sales, profit, orders, customer segments, and product categories
- Generated actionable insights to support business decision-making

## Tools Used

- Python
- Pandas
- Power BI
- Excel

## Business Impact

This dashboard provides a clear view of e-commerce sales performance and helps identify high-performing products, revenue trends, and customer behavior patterns. It can support better inventory planning, sales strategy, and data-driven business decisions.
